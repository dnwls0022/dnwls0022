## ✍️ About Me
> **"복잡한 데이터를 구조화하고, 사용자 중심의 직관적인 인터페이스를 설계합니다."** - **기록과 분석:** 생소한 기술 용어를 실생활에 비유하여 정리하며, 문제 해결 과정을 문서화하는 습관을 갖췄습니다. 
- **끈기 있는 완결성:** 정보처리기사 4번의 도전, 20회의 헌혈로 증명된 책임감으로 '업무 누수 제로'를 지향합니다. 
- **실무형 IT 인재:** 대동병원 자산관리팀에서의 실무 행정 경험과 풀스택 역량을 결합하여 현장에 즉시 투입 가능한 준비된 인재입니다.

## 📂 Featured Project
### 🎬 Connected_M: AI 통합 영화 탐색 및 분석 플랫폼
**"데이터 무결성과 사용자 경험의 완결성을 구현한 맞춤형 AI 솔루션"**

- **Architecture & Optimization:**
  - **Atomic Design 패턴** 적용을 통한 컴포넌트 모듈화 및 재사용성 극대화 
  - **React 기반 아키텍처 설계** 및 렌더링 최적화로 가독성 높은 인터페이스 구축
- **Data & Troubleshooting:** - 백엔드 API 분석 기반의 **TSX 컴포넌트 설계**로 데이터 정합성 유지 및 런타임 오류 최소화 
  - 프론트-백엔드 연동 과정의 병목 현상 및 통신 오류를 해결하여 안정적인 서비스 환경 구축 
- **AI Interactive UI:** - **Gemini AI 연동 모달창** 개발: 맞춤형 추천 키워드 생성 및 데이터 분석 인터페이스 제공
